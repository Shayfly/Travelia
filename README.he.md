# טרווליה

פרויקט חיפוש טיסות ומלונות עם React ו-Express.

## הרצה מקומית

```bash
# התקנת תלויות
cd frontend && npm install
cd ../backend && npm install

# הפעלת שרתי פיתוח
cd ../frontend && npm run dev
cd ../backend && npm run dev
```

## בנייה ופריסה

```bash
cd frontend
npm run build
npm run deploy # טעינת התיקייה dist ל-GitHub Pages
```
